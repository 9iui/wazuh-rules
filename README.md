# Wazuh Detection Engineering Repository

This repository contains detection rules, use cases, and rule tuning techniques developed as part of a structured detection engineering lab.

Focus areas include:
- Threat detection based on MITRE ATT&CK
- Log analysis and enrichment
- Detection logic development
- False positive tuning
- Alert severity design

Environment:
- Wazuh SIEM
- Sysmon
- Linux audit logs
- Windows event logs
- Active Directory lab
- Attack simulation

The purpose of this repository is to document detection engineering practices and continuously improve defensive visibility.
